Grails Test App for unit testing uri based filters
================================

This is a vanilla grails app.
Added:
 - UriFilters.groovy
 - UriController.groovy
 - UriFiltersTest.groovy
 
Installation
------------
Clone the repo

Usage:
------
run: grails run-app
visit: http://localhost:8080/uri-filter-unit-test/sec/uri/index

run: grails test-app
